программа с графическим интерфейсом дле работы с базой данных телеграм-бота из этого проекта https://github.com/yarrou/TelegramBot
возможности : поиск , добавление, редактирование и удаление городов в базе
настройка подключения осуществляется через переменную окружения URL_TLGRM_BT

a program with a graphical interface for working with the telegram bot database from this project https://github.com/yarrou/TelegramBot
features: search, add, edit and delete cities in the database
the connection is configured through the URL_TLGRM_BT environment variable
